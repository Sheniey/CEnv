## [*] About This:
#### CEnv:
***CEnv CE*** [Comiunity Edition] es una herramienta (Script) que crea **Entornos** de Desarrollo y Programacón *Customs*/*Predefinidos* por el usuario/cliente con una interfaz interactiva multilingüe [english, español]...

Si deseas una Edición más Light o Pro...
<fieldset>
    <legend>CEnv LE</legend>
    <p><b><i>CEnv LE</i></b> [Light Edition] es una Edición Reducida/Ligth de <i>CEnv CE</i> al tener una interfas estatica, ideal para <i>Optimizar Recursos</i> en el <b>Hardware</b>.</p>
</fieldset>
<br>
<fieldset>
    <legend>CEnv PE</legend>
    <p><b><i>CEnv PE</i></b> [Professional Edition] es una Edición Profesional/Pro de <i>CEnv CE</i>, ya que cuenta con una interfas <b><i>GUI</i></b> y una lista de <i>Entornos</i> Predefinidos enfocados para el uso en <b>Proyectos Profesionales</b>.</p>
    <blockquote>
        <p><b>[+] Advice:</b> Es Posible que en un futuro está <i>Edición</i> deje de ser <b>Gratis</b>.</p> 
    </nlockquote>
</fieldset>

#### Git Method:
[MacOS|Linux](#macoslinux "Goto |Download > ZIP Method > MacOS|Linux|") | [Windows](#windows-nt "Goto |Download > ZIP Method > Windows NT|")
###### **MacOS|Linux:**
1. Download ***CEnv*** from Bash:
```bash
$ git clone https://www.github.com/Sheniey/CEnv.git
```
2. Goto the ***CEnv** Folder:
```bash
$ cd "CEnv"
```
3. Escalate Privileges to ***cenv.py*** and *init-linux.sh* Files:
```bash
$ chmod +x cenv.py
$ chmod +x init-linux.py
```
> [!WARNING]
> You may not be able to Run ***CEnv*** if you do not Escalate Privileges on your *Operating System*.
4. Run *init-linux.sh* & Follow What It Asks You:
```bash
$ ./init-linux.sh
```
5. Finally, Run ***cenv.py*** for the First Time:
```bash
$ cenv.py
```

<br>

###### **Windows NT:**
> You Can Use ***pyintaller*** to Convert the *.py* to *.exe*. Just Move to *~\AppData\Local\Programs\CEnv\$edition*
1. Download ***CEnv*** from the Windows Terminal:
```powershell
> git clone "https://www.github.com/Sheniey/CEnv.git"
```
2. Goto the ***CEnv** Folder:
```powershell
> cd "CEnv"
```
3. Run *init-windows.bat* & Follow What It Asks You:
```powershell
> .\init-windows.bat
```
4. Finally, Run ***cenv.py*** for the First Time:
```powershell
> cenv
```
